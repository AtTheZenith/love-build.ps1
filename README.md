## About This Project

A multi-platform easy-to-install build script for LÖVE developers who want quick `.love` files and standalone binaries for Windows and Linux.

> [!NOTE]
> Linux is not supported at the moment, this will be looked into in the near future.

---

## Requirements

- **Git**  
- **PowerShell 7 and up**

## Features

- Automatically builds `.love` files from your project  
- Generates standalone **Windows executables** and **Linux AppImages**  
- Includes **autorun** (auto-launch)  
- Uses **Powershell 7** for multi-platform support (soon)  
- Configurable LÖVE version and build options
